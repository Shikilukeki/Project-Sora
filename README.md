## Deskripsi Aplikasi
Sora merupakan aplikasi music player lokal yang berfokus pada kemudahan penggunaan dan pengalaman pengguna (UI/UX).  
Aplikasi ini memungkinkan pengguna untuk memutar musik secara offline langsung dari penyimpanan perangkat, serta dilengkapi dengan beberapa fitur tambahan untuk meningkatkan personalisasi dan interaksi pengguna.

---

## Fitur Aplikasi
- Pemutar musik lokal (play, pause, next, previous)
- Pencarian musik
- Mini music player
- Riwayat lagu yang diputar
- Randomize / AI Pick Music
- Deteksi lokasi pengguna dan greeting (Say Hello) sesuai lokasi
- Pengubahan tema aplikasi (Light & night mode)
- Pengaturan bahasa (Bahasa Indonesia & Bahasa Inggris)
- Edit username pada profil (sapaan di halaman utama)
- Notifikasi menggunakan Firebase
- Notifikasi kontrol musik (play/pause) melalui notification bar

---

## Teknologi yang Digunakan
- Android Studio
- Bahasa Pemrograman: Java
- Firebase (Notification)
- Local Storage (Media file dari perangkat)
